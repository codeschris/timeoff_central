# Timeoff Central

Welcome to the Timeoff Central repository. This project is designed to manage and track employee time off requests efficiently.

## Features

- User authentication and authorization
- Request and approval leaves
- Calendar integration
- Notifications and reminders

## Tools/Technologies Used

- **Next.js (Typescript)** - Frontend framework for building the user interface
- **Django** - Backend framework for handling server-side logic and database interactions
- **Shadcn/ui** - UI component library for consistent and reusable UI elements

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/yourusername/timeoff_central.git
    ```

2. Install dependencies for the frontend:

    ```bash
    cd frontend
    npm install
    ```

3. Install dependencies for the backend:

    ```bash
    cd backend
    pip install -r requirements.txt
    ```

4. Start the development servers:
    - Frontend:

        ```bash
        npm run dev
        ```

    - Backend:

        ```bash
        python manage.py runserver
        ```
